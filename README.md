# Data Analysis in R

This repository contains materials for an introductory **Data Analysis in R** workshop presentation.

## Main Files

- `R_part1.html` - the rendered presentation. This is the file to open in a web browser when viewing or presenting the slides.
- `R_part1.qmd` - the Quarto source file used to generate the presentation. Edit this file when changing the slide content.
- `styles.css` - custom CSS used by the presentation.

## Supporting Folders

- `figures/` - images and graphics used in the slides.
- `R_part1_files/` - supporting JavaScript, CSS, fonts, and Reveal.js assets generated by Quarto when `R_part1.html` is rendered.

## Important Note About the HTML and QMD Files

The `.html` file is the actual presentation output. It can be opened directly in a browser and is what participants or presenters should use to view the slides.

The `.qmd` file is the editable Quarto script/source file. It contains the slide text, image references, and presentation settings. After editing `R_part1.qmd`, render it again to update `R_part1.html`.

## Sharing or Publishing

If sharing the rendered presentation, keep these together:

- `R_part1.html`
- `R_part1_files/`
- `figures/`
- `styles.css`

Without the supporting folders and CSS file, the HTML presentation may not display correctly.
